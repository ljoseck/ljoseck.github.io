This custom 3D renderer was made in an afternoon as a quick personal project and rapid prototype. This project also served as a challenge in programming and mathematics as I limited myself to no online resources leaving me to work out the rendering equations needed with paper and pencil. I also created this with the intension of making interesting 3D shapes and animations. I am not much of an artist, but I enjoy the geometric designs people have made.

This project was done with JavaScript on an HTML canvas. JavaScript was chosen as I could easily render 2D lines and shapes with animations from different calculated frames. HTML allows for this project to be easily demoed and shared. By selecting the main.html file below you can view the final renderer in action.

The renderer I made is a single point perspective renderer meaning that objects very close to the camera would appear distorted and curved. While my renderer could achieve these distortions by plotting each point on a line, I had to make assumptions to improve performance. Each line in the 3D space would appear as straight lines on the two-dimensional canvas, while maintaining the correct end points. All the calculations to create the core renderer with a stationary camera were done independently in the same afternoon I started this project. This rapid prototype was able to render any three-dimensional shape as a wire frame.

My next goal was allowing for a moving camera. While I created the high-level functionality to achieve this goal, I did get help online concerning a certain function allowing for the rotation of the camera. Knowing what I was looking for I discovered a function that would rotate an arbitrary point some angle about an arbitrary vector. This allowed for the full wire frame animation that is currently demoed.
